# Pilates By Ela Website

Bu proje, HTML, CSS ve JavaScript kullanılarak hazırlanmış Pilates salonu tanıtım ve sağlıklı beslenme web sitesidir. Web sitesinde Pilates By Ela markası için ana sayfa, üyelik tanıtımı, görsel içerik alanları ve sağlıklı besin kategorilerinin listelendiği beslenme sayfası bulunmaktadır.

## Proje Amacı

Bu projenin amacı, bir Pilates salonu için sade, görsel ağırlıklı ve kullanıcı dostu bir tanıtım web sitesi oluşturmaktır. Ana sayfada salonun genel tanıtımı yapılırken, beslenme sayfasında sağlıklı yiyecek, içecek ve salata önerileri kategori bazlı olarak gösterilmektedir.

## Kullanılan Teknolojiler

* HTML
* CSS
* JavaScript
* Font Awesome
* Google Fonts



## Sayfalar

### index.html

Projenin ana sayfasıdır. Bu sayfada Pilates By Ela başlığı, logo, menü alanı, büyük arka plan görseli, slogan, tanıtım metni, yaşam/güç/değişim görsel bölümleri ve üyelik açıklaması yer almaktadır.

Ana sayfada bulunan başlıca bölümler:

* Header alanı
* Logo ve marka başlığı
* Beslenme, Alışveriş ve Kurslar menüsü
* Büyük görsel arka plan
* Slogan alanı
* Tanıtım metni
* Yaşam, Güç ve Değişim kartları
* Üyelik tanıtımı

### beslenme.html

Beslenme sayfası, kullanıcıların sağlıklı besin kategorilerini görüntülemesini sağlar. Sayfada üç farklı kategori bulunmaktadır:

* Yiyecek
* İçecek
* Salata

Kullanıcı kategori butonlarına tıkladığında ilgili besinler JavaScript ile dinamik olarak ekrana gelir.

## JavaScript Özelliği

`script.js` dosyasında kategori filtreleme mantığı bulunmaktadır. Kullanıcı Yiyecek, İçecek veya Salata butonuna tıkladığında, seçilen kategoriye ait besinler sayfada kart yapısı ile gösterilir.

Her kartta şu bilgiler yer alır:

* Besin adı
* Görsel
* Kalori bilgisi
* Kısa açıklama

Örnek kategoriler:

* Avokado
* Kinoa
* Somon
* Yeşil Çay
* Kefir
* Avokado Salatası
* Kinoa Salatası

## CSS Yapısı

Projede iki farklı CSS dosyası kullanılmıştır.

### style.css

Ana sayfanın tasarımını içerir. Header, arka plan görseli, menü animasyonu, tanıtım yazısı, üçlü görsel alan ve üyelik bölümü bu dosya üzerinden stillendirilmiştir.

### bstyle.css

Beslenme sayfasının tasarımını içerir. Kategori butonları, besin kartları, kart hover efektleri ve beslenme sayfasının genel görünümü bu dosya üzerinden düzenlenmiştir.

Projede kullanılan görseller `RESİMLER` klasörü içerisinde tutulmaktadır. Sitenin doğru çalışması için bu klasörün de proje ile birlikte yüklenmesi gerekir.

Örnek görseller:

* Logo görseli
* Ana sayfa arka plan görseli
* Pilates görselleri

* İçecek görselleri
* Salata görselleri

Bu proje statik bir web sitesidir. Herhangi bir backend veya veritabanı gerektirmez.

Projeyi çalıştırmak için `index.html` dosyasını tarayıcıda açmak yeterlidir.
